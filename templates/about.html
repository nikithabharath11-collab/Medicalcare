Project Overview

This project demonstrates the application of Machine Learning and Deep Learning techniques in the medical and healthcare domain. It features multiple AI models for disease prediction, risk assessment, and medical image analysis, all served through an interactive web interface.

Disclaimer: This is an educational project. Predictions should NOT be used as a substitute for professional medical advice, diagnosis, or treatment.

Key Features

Heart Disease Prediction

Gradient Boosting Classifier with 13 clinical features for cardiovascular risk assessment.

Diabetes Prediction

Random Forest Classifier with 8 health indicators for diabetes risk analysis.

Disease from Symptoms

Multi-class Random Forest with 20 symptoms covering 10 diseases.

Breast Cancer Detection

Deep Neural Network (TensorFlow/Keras) with 10 cell nucleus measurements.

Medical Image Classification

CNN with MobileNetV2 Transfer Learning for chest X-ray analysis.

AI Health Chat

Google Gemini powered chatbot for instant medical guidance with rule-based fallback.

BMI Calculator

Body Mass Index calculator with personalized health tips and PDF reports.

User Auth & History

Login/Register system with prediction history tracking in SQLite database.
